# piggybank
a virtual piggy bank app

## frontend app
```
$ cd reactapp
$ yarn dev
```

### To deploy
```
$ yarn build
$ yarn start
```

## backend server
first time only, to create a virtual env with python3: `python3 -m venv env`
```
$ cd server
$ source env/bin/activate
$ python manage.py runserver localhost:8000
```
